# Patient review task

1. Read the patient demographics and workflow state.
2. Compare diagnoses, medications, labs, and encounter notes.
3. Compare the documentation with only the supplied payer or workflow rules.
4. Identify missing, conflicting, or insufficient information.
5. Recommend one specific, reviewable action for each supported issue.
6. Return no finding when the supplied evidence does not support one.

All recommendations require human approval before any patient or workflow state changes.
